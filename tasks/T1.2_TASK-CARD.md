## 五、技术设计

### 5.1 文件路径约定

```
cycles/
├── cycle_unit.schema.yaml        # Schema 定义（T1.1）
├── {scope}/                       # scope 子目录
│   └── {id}.yaml                  # CycleUnit 文件
└── archive/                       # 归档目录（T1.3）
    └── {scope}/
        └── {id}_v{version}.yaml   # 历史快照
```

### 5.2 核心函数设计

```python
def load_cycle(id: str, scope: str) -> dict:
    """加载 CycleUnit
    
    Args:
        id: CycleUnit ID（格式：{scope}-{timestamp}-{random}）
        scope: 执行粒度（task/topic/project/system）
    
    Returns:
        CycleUnit dict，包含所有字段
    
    Raises:
        FileNotFoundError: 文件不存在
        yaml.YAMLError: YAML 格式错误
    """
    path = f"cycles/{scope}/{id}.yaml"
    with open(path, 'r') as f:
        return yaml.safe_load(f)

def save_cycle(unit: dict) -> None:
    """原子性保存 CycleUnit
    
    Args:
        unit: CycleUnit dict
    
    Features:
        - 自动递增 version
        - 自动更新 updated_at
        - 原子性写入（.tmp + rename）
    """
    # 自动更新字段
    unit['metadata']['version'] += 1
    unit['metadata']['updated_at'] = datetime.utcnow().isoformat()
    
    # 原子性写入
    path = f"cycles/{unit['scope']}/{unit['id']}.yaml"
    tmp_path = path + '.tmp'
    
    with open(tmp_path, 'w') as f:
        yaml.dump(unit, f)
    
    os.rename(tmp_path, path)  # 原子操作

def update_phase(id: str, scope: str, new_phase: str) -> None:
    """更新 phase 并归档旧版本
    
    Args:
        id: CycleUnit ID
        scope: 执行粒度
        new_phase: 新阶段（plan/do/check/act）
    
    Features:
        - 归档旧版本到 archive/
        - 更新 phase 字段
        - 触发 save_cycle()
    """
    unit = load_cycle(id, scope)
    old_version = unit['metadata']['version']
    
    # 归档旧版本
    archive_path = f"cycles/archive/{scope}/{id}_v{old_version}.yaml"
    shutil.copy(f"cycles/{scope}/{id}.yaml", archive_path)
    
    # 更新 phase
    unit['phase'] = new_phase
    save_cycle(unit)
```
