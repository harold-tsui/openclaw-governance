# Review Report 模板

> **用途**：交付物审查报告
> **版本**：v1.0
> **所属 Skill**：openclaw-governance-quality

## 使用说明

1. 任务提交验收后，Reviewer 创建此报告
2. 逐一验证 DOD 标准
3. 记录发现的问题
4. 给出审查结论
