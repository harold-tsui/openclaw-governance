# TRAINING-SOP · Agent 培训流程

> **版本**：v1.0
> **触发条件**：Skill 变更/能力升级需求/定期培训
> **审批级别**：L1/L2
> **预计耗时**：1-4 小时（视培训类型）

---

## 一、培训类型

| 类型 | 触发 | 审批 | 预计耗时 |
|------|------|------|----------|
| **Skill 培训** | Skill 安装/更新 | L1 | 1-2 小时 |
| **能力升级培训** | 需求申请 | L2 | 2-4 小时 |
| **定期培训** | 季度/年度 | L1 | 1-2 小时 |

---

## 二、流程概览

```
Step 1: 培训需求识别 → Step 2: 培训执行 → Step 3: 能力验证 → Step 4: 能力档案更新
```

---

## 三、详细步骤

### Step 1: 培训需求识别

```
□ 识别培训触发条件
□ 确定培训内容（Skill/能力）
□ 制定培训计划
```

---

### Step 2: 培训执行

```
□ 学习相关文档/Skill
□ 完成练习任务
□ 记录学习笔记
```

---

### Step 3: 能力验证

```
□ 完成验证任务
□ 输出培训报告
□ 提交银月审核
```

**交付物**：`TRAIN-REPORT-{agent_id}-{date}.md`

---

### Step 4: 能力档案更新

```
□ 更新 agents.yaml 能力清单
□ 记录培训历史到个人档案
□ 通知 Harold（如涉及重要能力）
```

---

## 四、审批矩阵

| 培训类型 | 审批人 | 审批级别 |
|----------|--------|----------|
| Skill 培训 | 银月 | L1 |
| 能力升级培训 | 银月 + Harold 知情 | L2 |
| 定期培训 | 银月 | L1 |
